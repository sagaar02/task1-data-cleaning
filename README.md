# task1-data-cleaning
## Project Overview
This project focuses on cleaning and preprocessing a mall customer dataset using Python and Pandas.  
The goal is to prepare raw data for further analysis and machine learning tasks.

---

## Objectives
- Handle missing values
- Remove duplicate records
- Fix inconsistent formatting
- Standardize column values
- Prepare structured data for analysis

---

## Tools & Technologies
- Python
- Pandas
- Google Colab

---

## Tasks Performed
### Data Cleaning Steps
- Checked dataset structure
- Identified missing values
- Removed duplicates
- Fixed formatting inconsistencies
- Renamed columns where necessary
- Verified cleaned dataset quality

---

## Learning Outcomes
- Gained hands-on experience in data cleaning
- Improved understanding of preprocessing techniques
- Learned practical usage of Pandas functions
- Built confidence in handling raw datasets

---

## Dataset
Mall Customer Dataset used for preprocessing and cleaning practice.
